# 🌤️ Weather Detection App

A modern and responsive weather web application that allows users to search for any city and get real-time weather data, including temperature, humidity, wind speed, and more.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature display (°C). (Here, we take the api key from OpenWeather.com)
* ☁️ Weather condition & icon
* 💧 Humidity, pressure, and wind speed
* 👁️ Visibility and "feels like" temperature
* 🌅 Sunrise & 🌇 sunset timings
* ⚡ Fast and responsive UI
* ❌ Error handling for invalid city/network issues
* 🔄 Live loading spinner

---

## 🧑‍💻 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **API:** OpenWeatherMap API
* **Fonts:** Google Fonts (DM Sans, DM Serif Display)

## 🔑 API Setup

This project uses the **OpenWeatherMap API**.

1. Go to: https://openweathermap.org/api
2. Create a free account
3. Generate your API key
4. Replace this line in your code:

```JavaScript
const API_KEY = 'your_api_key_here';
```
